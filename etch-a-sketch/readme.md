# Etch-A-Sketch

A simple web-based grid drawing app inspired by the Etch-A-Sketch. Draw by hovering or tapping cells and resize the grid.  

## Live Demo
[LiveDemo](https://tantodelight.github.io/webdev-playground/etch-a-sketch/)

## How to Use
- Click **Resize grid** to change the grid size.  
- Hover or tap on cells to draw.
